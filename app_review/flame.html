Hunger Games

This application is a sentiment analysis tool for book reviews. It is designed for The Hunger Games. The technology behind this app is named Naïve Bayes Classifier.


What's behind the scene?



This application makes classifications with a Naïve Bayes Classifier described in [1]. Data are reviews of The Hunger Games. It is gathered manually through Amazon Book Store and preprocessed in NB.js. Words and their occurrences are stored in a memory named wordBank. When the user types a review, the model uses wordBank to compute the probabilities.

[1] Jurafsky, D., & Martin, J. H. (2018). Speech and language processing.
